# Select

A input that allows to select between several options.

```html
<html>
  ...
  <body>
    <select>
      <option>A</option>
      <option>B</option>
      <option>C</option>
    <select>
    ...
  </body>
</html>
```

## Children

- [Option](option.md)

## Attributes

None
