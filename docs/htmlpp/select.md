# Select

A input that allows to select between several options.

```html
<html>
  ...
  <body>
    <select>
      <option>A</option>
      <option value="T">B</option>
      <option disabled>C</option>
    </select>
    ...
  </body>
</html>
```

## Children

- [Option](option.md)

## Attributes

- Disabled - whether the select can be changed
