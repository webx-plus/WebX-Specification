# Option

A option inside a select.

```html
<html>
  ...
  <body>
    <select>
      <option>A</option>
      <option value="T">B</option>
      <option disabled>C</option>
    </select>
    ...
  </body>
</html>
```

## Content

- Text

## Attributes

- value - it's value
- disabled - whether the option can be selected
