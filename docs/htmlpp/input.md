# Input

A one line text input.\
Self-closing.

```html
<html>
  ...
  <body>
    <input type="text" placeholder="Hi">
    <input type="number" placeholder="Locked" disabled>
    ...
  </body>
</html>
```

## Attributes

- placeholder - placeholder text
- type - the type of input
- disabled - whether the input can be written to
