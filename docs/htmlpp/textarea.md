# Textarea

A multi line text input.\
Self-closing.

```html
<html>
  ...
  <body>
    <textarea placeholder="Write a lot">
    ...
  </body>
</html>
```

## Attributes

- placeholder - placeholder text
- disabled - whether the textarea can be written to
