# Audio
A audio player.\
Self-closing.\
Note: Unlike html, the controls are always shown.

```html
<html>
  ...
  <body>
    <audio src="<url>">
    ...
  </body>
</html>
```

## Content
None

## Attributes
- Class - classes it has
- Src - the source of the audio
