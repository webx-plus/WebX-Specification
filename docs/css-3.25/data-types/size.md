# \<size>

A type that defines a length, size or dimension value.

```css
<size>
12px
```

{: .note }
Some browsers allow properties that require sizes to have no unit attached and will default to pixels, this should not be relied upon.

## Units

- [px](../units/px.md)
- [pt](../units/pt.md)
