# Padding-\[side]

Defines the padding/inner separation for a specific side applied to an element.

```css
padding-[side]: <size>;
padding-top: 12px;
```

## Syntax

`padding-[side]: <size>;`

Side must be one of: top, right, bottom, or left.\
Value must be of type [\<size>](../data-types/size.md).
