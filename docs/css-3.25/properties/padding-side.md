# Padding-\[side]

Defineds a padding for a specific side applied to this element.

```css
padding-[side]: <size>;
padding-top: 12px;
```

## Syntax

`padding-[side]: <size>;`

Side can be top, right, bottom or left.\
Value must be of type [\<size>](../data-types/size.md).
