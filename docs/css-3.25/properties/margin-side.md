# Margin-\[side]

Defineds a margin or separation for a specific side applied to this element.

```css
margin-[size]: <size>;
margin-top: 12px;
```

## Syntax

`margin-[size]: <size>;`

Side can be top, right, bottom or left.\
Value must be of type [\<size>](../data-types/size.md).
