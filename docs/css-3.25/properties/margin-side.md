# Margin-\[side]

Defines the margin/separation for a specific side on an element.

```css
margin-[side]: <size>;
margin-top: 12px;
```

## Syntax

`margin-[side]: <size>;`

Side must be one of: top, right, bottom, or left.\
Value must be of type [\<size>](../data-types/size.md).
