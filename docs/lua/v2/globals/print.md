# Lua v2 Print

This global function prints text passed to it.

```lua
print('Hello')
```

## Inputs

- Text - any text to print

## Return

No return.

## Support

|       | YAB  | WXV  | WXB  |
| ----- | :--: | :--: | :--: |
| print | Full | Full | Full |
| text  | Full | Full | Full |
