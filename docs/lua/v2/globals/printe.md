# Lua v2 PrintE

This global function prints a error.

```lua
printe('Incorrect response')
```

## Inputs

- Text - any text to print

## Return

No return.

## Support

|        | YAB  | WXV  | WXB  |
| ------ | :--: | :--: | :--: |
| printe | Full | Full | Full |
| text   | Full | Full | Full |
