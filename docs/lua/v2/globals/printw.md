# Lua v2 PrintW

This global function prints a warning.

```lua
printw('Need more trees')
```

## Inputs

- Text - any text to print

## Return

No return.

| ------ | :--: | :--: | :--: |
| printw | Full | Full | Full |
| text   | Full | Full | Full |
