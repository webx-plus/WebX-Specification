# DNS

This serves as a few required api endpoints for the dns, it may have more.

## Endpoints

### Resolve

GET `/domain/<domain>/<tld>`\
The IP the domain points to

```json
{ "ip": "https://..." }
```

## Schemas

### Domain

```json
{ "ip": "https://..." }
```

IP: A URL or IP address the domain should point to
